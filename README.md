# Student Grade Management System (SGMS)

A simple, efficient, and user-friendly console-based application to manage students, subjects, record grades, and generate performance reports — built using pure Python.

## Project Title
**Student Grade Management System (SGMS)**

## Overview of the Project
This system helps teachers/academic staff maintain student records and grades without relying on spreadsheets. It supports adding students & subjects, recording marks, and generating detailed reports with pass/fail status and statistics.

## Features
- Add, view, and delete students
- Add and view subjects
- Record and validate grades (0–100)
- Generate complete grade report with pass/fail status
- Automatic data persistence using JSON
- Backup creation on every save
- Clean, menu-driven interface

## Technologies/Tools Used
- Python 3.8+
- Standard Library only (`json`, `os`, `datetime`)
- No external dependencies
